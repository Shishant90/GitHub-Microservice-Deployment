## 🚀 Quick Start

### Ubuntu Setup (Recommended)
```bash
# Easy setup
chmod +x setup.sh
./setup.sh

# Run application
chmod +x run.sh
./run.sh
# Visit: http://localhost:5000
```

### Alternative Setup
```bash
# Manual setup
sudo apt-get update
sudo apt-get install -y python3-full python3-pip python3-venv
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run application
cd ci-cd-pipeline-project
python src/app.py
```

### Docker Deployment
```bash
make docker-run
```
